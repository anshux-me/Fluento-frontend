# LinguaAI Frontend

React + Vite + Tailwind frontend with brutalist minimal design.

## Features

- **Firebase Auth**: Email/password and Google login
- **Gamification**: XP, levels, streaks, badges
- **Audio Recording**: Browser-based microphone recording
- **Responsive**: Mobile-friendly brutalist design

## Quick Start

```bash
# Install dependencies
npm install

# Copy environment file
cp .env.example .env
# Edit .env with your Firebase config

# Run development server
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Push to GitHub
2. Import on Vercel
3. Set environment variables
4. Deploy
